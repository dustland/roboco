Quick Start

Get started with AgentX in just a few minutes:

pip install agentx
from agentx import Agent, TaskExecutor
 
# Create a simple agent
agent = Agent(
    name="assistant",
    model="gpt-4",
    system_prompt="You are a helpful assistant"
)
 
# Execute a task
executor = TaskExecutor("path/to/config.yaml")
result = await executor.run("Hello, how can you help me?")
print(result)
